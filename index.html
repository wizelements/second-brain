What's the agent?

The Windows Brain Agent is a background service that listens for commands from Termux and executes them on Windows. It runs on port 8080.

Why start it?

‚úì Termux can send commands to Windows
‚úì Remote file operations work
‚úì Cross-device automation enabled

How long to start?

Usually 5-10 seconds. You'll see "Agent Running" here when ready.

Make It Auto-Start

Run this in PowerShell as Admin to auto-start on Windows boot:

$taskName = "Windows Brain Agent"
$action = New-ScheduledTaskAction -Execute "powershell.exe" -Argument '-NoProfile -WindowStyle Hidden -File "C:\Users\jacla\Scripts\windows-brain-agent-v4.ps1"'
$trigger = New-ScheduledTaskTrigger -AtStartup
Register-ScheduledTask -TaskName $taskName -Action $action -Trigger $trigger -Force
